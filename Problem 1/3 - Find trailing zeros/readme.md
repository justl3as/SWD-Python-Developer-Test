### [Problem 1.3]

> เขียนโปรแกรมหาจำนวนเลข 0 ที่อยู่ติดกันหลังสุดของค่า factorial ด้วย Python โดยห้ามใช้ math.factorial เช่น 7! = 5040 มีเลข 0 ต่อท้าย 1 ตัว, 10! = 3628800 มีเลข 0 ต่อท้าย 2 ตัว

--

Example 1:

```
Input: number = 0
Output: 0
```

Example 2:

```
Input: number = 7
Output: 1
```

Example 3:

```
Input: number = 50
Output: 12
```
